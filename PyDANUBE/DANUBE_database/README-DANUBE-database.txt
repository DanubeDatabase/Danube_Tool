DANUBE Database
Données Archétypales aux Niveaux Urbain et Bâtiment pour des Evaluations-environnementales

DANUBE is an architectural database which provides materiality informations from archetypal knownledge and sereval input data (building typology, usage, construction period and location).

The DANUBE database is built from the "Base de données architecturale MApUCE" initially developped by the research laborotory LRA-ENSA-Toulouse, during the MApUCE ANR research project (see https://lra.toulouse.archi.fr/lra/activites/projets/mapuce ). The database has been enhanced during PAENDORA2 Ademe's research project (see https://lra.toulouse.archi.fr/lra/activites/projets/paendora2/) for the development of the GIS plugin "DANUBE tool" wich embed the DANUBE database.
